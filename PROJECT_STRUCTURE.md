## How to Use This Structure

### For Development
1. Check the [Project Board](https://github.com/users/SamuelHauptmannvanDam/projects/7) for current sprint
2. Filter by milestone to see what's coming up
3. Use priority labels to determine what to work on first

### For Planning
1. View [Milestones](https://github.com/SamuelHauptmannvanDam/findkandidat/milestones) to see progress toward releases
2. Use the Project Board's roadmap view to visualize timeline
3. Review issues by label to see what areas need attention

1. Check issues labeled `priority-high` in the current milestone
2. Look for issues matching your expertise (frontend, backend, database, etc.)
3. Comment on issues to get assigned
